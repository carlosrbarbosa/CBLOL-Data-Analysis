# CBLOL Data Analysis

Análise de dados competitivos do CBLOL (Campeonato Brasileiro de League of Legends) utilizando Python, Pandas e visualização de dados.

## Objetivo
Explorar dados históricos do CBLOL para identificar padrões de desempenho de equipes, jogadores e metas competitivas ao longo das temporadas.

## Dados
Os dados foram coletados de fontes públicas de e-sports (Leaguepedia / Oracle Elixir).

Incluem:
- Resultados de partidas
- Estatísticas de jogadores
- Estatísticas de equipes
- Ouro, kills, torres, duração
- Posição e função

## Pipeline de dados
1. Coleta de dados (scraping/API)
2. Limpeza e tratamento (Pandas)
3. Análise exploratória (EDA)
4. Criação de métricas
5. Visualização

## Análises realizadas
- Win rate por equipe
- Desempenho por função
- KDA médio por jogador
- Evolução por temporada
- Correlação ouro vs vitória
- Tempo médio de partidas

## Tecnologias
- Python
- Pandas
- Matplotlib / Seaborn
- Jupyter
- SQL (opcional)

## Insights
- Equipes com maior consistência histórica
- Funções com maior impacto em vitória
- Relação entre economia e resultado
- Metas dominantes por temporada
